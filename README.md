# TaniK

## About

This project is an **RPG-style game** built from the ground up using Python.

### Tech Stack

* **Language:** Python 3.14+
* **Genre:** RPG (Role-Playing Game)

## How to Run (Linux / macOS)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/FaSTIK6/TaniK.git
   cd TaniK
   ```

2. **Sync dependencies:**
   ```bash
   uv sync
   ```

3. **Go to src directory:**
   ```bash
   cd src
   ```

4. **Run the game:**
   ```bash
   uv run python -m tanik.main
   ```
