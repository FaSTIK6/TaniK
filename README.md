# TaniK

## About

This project is an **RPG-style game** built from the ground up using Python.

### Tech Stack

* **Language:** Python 3.14
* **Genre:** RPG (Role-Playing Game)

## How to Run

1. Clone the repository:
   `https://github.com/FaSTIK6/TaniK.git`
2. Sync dependencies:
   `uv sync`
3. Run the game:
   `uv run src/tanik/main.py`
